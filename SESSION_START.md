# SESSION_START.md

## 목적

새 세션이 시작될 때, 현재 프로젝트와 운영 체계를 빠르게 파악할 수 있도록 우선 참조 문서를 안내하는 시작 가이드다.

## 기본 원칙

- 새 세션은 이전 대화 전체를 신뢰하지 말고, 기록된 문서를 기준 원본으로 삼는다.
- 기본 성격과 사용자 정보는 워크스페이스 루트 문서를 따른다.
- 프로젝트 관련 판단은 engineering-playbook 문서를 우선 참조한다.
- 개인 기록과 운영 메모는 Obsidian vault를 참조한다.

## 사용 방법

- 새 프로젝트 시작 전, 위 문서 중 현재 작업과 직접 관련된 것부터 읽는다.
- 문서와 이전 대화 기억이 충돌하면 문서를 우선한다.
- 필요한 경우 Bot Logs에서 실제 요청/응답 원문을 확인한다.
- 새 세션에서 긴 설명보다 현재 관련 문서 요약을 먼저 만든 뒤 작업에 들어간다.

## 결론

새 세션은 대화를 이어받는 방식보다 문서를 다시 읽고 재앵커링하는 방식으로 시작하는 것이 안전하다. 이 문서는 그 출발점이다.
